# Kproject

kproject is initializing tool for Machine Learning task.

## Usage
### Install

```
$ pip install git+
```


```
$ kproject new mnist
[ CREATE ] New project: `mnist`
$ cd mnist; tree
.
├── README.md
├── dataset
├── log
├── model
├── result
└── src
    ├── lib
    │   └── args.py
    └── main.py

5 directories, 4 files
```
